\begin{itemize}

    \item \textbf{Python}: 3.10

\subsection{Evaluation Methodology}
\label{subsec:evaluation-methodology}

Evaluating an AI tutoring system for mathematics is inherently challenging
because correctness requires verifying both mathematical validity and
stylistic adherence. We employ a multi-dimensional evaluation approach.

\subsubsection{Evaluation Dimensions}

We evaluate each system variant along four dimensions:

\begin{enumerate}
    \item \textbf{Mathematical correctness}: Does the answer contain correct
    mathematical reasoning and computations?

    \item \textbf{Curriculum fidelity}: Does the answer stay within the
    Tunisian Bac program boundaries, using only permitted methods and
    theorems?

    \item \textbf{Style adherence}: Does the answer follow the Tunisian
    correction redaction conventions (formal French phrasing, structured
    proof layout)?

    \item \textbf{Retrieval quality} (RAG and hybrid only): Are the
    retrieved documents relevant to the query? Do retrieval scores reflect
    actual relevance?
\end{enumerate}
